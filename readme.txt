== Description ==

KnB Lite – Wordpress Knowledge Base / FAQ Plugin gives you control on your knowledge base items. You can easily assign categories and display it on up to Four(4) Columns in any part of your website. Easily create shortcode using the plugin’s powerful shortcode generator with options to display per category, orderby title or date, controls visibility and assign layout options.

<h4>Plugin Features</h4>
<ul>
<li><strong>Shortcode Generator</strong></li>
<li><strong>Search Form</strong></li>
<li>Display up to <strong>Four(4) Columns</strong></li>
<li>Custom post type and category slug Options</li>
<li><strong>Post Format</strong> icon support</li>
<li>Fully Responsive</li>
<li>Customize the display to match your current theme! Choose your own color style :)</li>
<li>Browsers Compatible</li>
<li>Powerful and Easy Settings Options</li>
</ul>

<h4>How to Use</h4>
<ol>
<li>Install and Activate the Plugin</li>
<li>Go to Knowledge Base > Shortcode Generator</li>
<li>Select and fill up any options</li>
<li>Generate the Shortcode</li>
<li>Copy and Paste the Shortcode to any posts, pages or part of your wordpress site</li>
</ol>
